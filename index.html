This website displays the locations Big Wands that can be spawned in various ways. Such wands are occasionally useful for obscure builds that require more than 26 slots.

The Great Chest wands can be obtained by getting a great chest to spawn at exactly the given coordinates. Usually this is done with the End of Everything. Taikasauva wands are spawned with Summon Taikasauva. Tiny drops check the tier 10 non-shufflers dropped when Limatoukka is killed. The coordinates record the pixel that Tiny should die on.

The farland button sets the search to locations where coordinate rounding allows for a larger than 1-pixel region to align the chest. This only affects chests, and not taikasauva or Tiny drops.
